다음 요구사항을 구현하기위해 사용할 optimize, pack 알고리즘을 고안하라.
오류가 절대 없어야하며, 빠른 성능이 보장되어야한다.

---

다음과 같이 작동하는 python script를 `/scripts` 경로에 .py 파일로 생성하라.

- selenium을 사용한다.
- 해당 주소의 페이지를 클론하기위한 디자인 가이드를 추출하는 것이 목표다.

프로세스:

1. 실행 시 command line에서 '추출할 웹페이지 주소'를 입력받는다.
2. 브라우저를 실행하고 해당 주소로 이동한다. 추출할 준비가 되면
   Enter를 입력하라고 안내한다.
3. Enter가 입력되면 해당 시점에 브라우저에 띄워진 페이지 내용을
   추출한다. HTML을 전부 렌더링한 뒤에, 관련된 css도 추출한다.
4. optimize: 추출된 HTML에서 디자인 가이드 분석과 무관한 내용을 모두 제거한다.
5. pack: 관련 파일을 모두 합친 output.xml 파일을 생성한다.
